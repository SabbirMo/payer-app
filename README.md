## 🚀 HOW TO BUILD YOUR APK/AAB:

### Step 1: Install Flutter
→ Go to flutter.dev → Download → Install

### Step 2: Get packages
```
flutter pub get
```

### Step 3: Build for Google Play Store
```
flutter build appbundle --release
```
Your AAB file: build/app/outputs/bundle/release/app-release.aab

### Step 4: Or build APK directly
```
flutter build apk --release
```
